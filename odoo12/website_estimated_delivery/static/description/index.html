Website Estimated Delivery

Website Estimated Delivery allows your customers to know the date of the delivery of their ordered products. You can set the days within which the product will be available for a particular zipcode. Multiple zipcodes are seperated by commas(,). Estimated delivery days can be calculated using two ways one is by using default delivery lead time and other by using pincodes.

Salient Features
  • This module provides the features to the website users to check the mimimum number of days within which the product will be delivered based on the pincodes..
  • Specify a range of comma seperated zip codes and the days in which the product is available in those zip codes.
  • Estimated delivey can be meausred using two methods.One is using Default Delivery method which is based on the default lead time and the second is Advanced Delivery which is based on the zip codes.
  • Show/Hide the estimated delivery option in the cart page on your choice.
  • Show/Hide the estimated delivery option in the product page on your choice.
  • The days within the product is available can de dispalyed in two ways. One is using exact number of days(eg, 10days) and the second is range of days(eg, 5-7days)
  • Choose the messages to be displayed in website on your choice.
  • Global configuration can be overriden product wise. Specify a new set of values and a new set of zip codes per product.
  • It works also for the public users.
  • It has many more features described below.

CONFIGURATIONS

Global Configuration

Some Important Points.

  • Show delivery time in Product page.
  • Show delivery time in cart page.
  • Message to display when the product is available
  • Message to display when the product is unavailable.
  • Display Mode(How to display the days in website(Exact number of days or Range of days).
  • Compute delivery time using(Default Delivery Lead Time based on default lead time/ Advanced Delivery Time based on pincodes)

Some Important Points

  • Create Delivery Day Range.

    The days within a product is availabe in webiste will be displayed as a range of days(eg, 5-7 days).

  • Number of days(postponed or preponed).

    Note:- These Fields will be visible only when the Display mode selected is Range.The Number of days will be added or subtrcted from the actual(exact) days in order to create the range of days.

Some Important Points.

  • Comma Seperated Zipcodes,

    Specify the comma seperated zip codes

  • Days Within Product will be available.

    Zipcodes can be enterd only when the Compute Delivery Time Using is choosen as Advanced Estimated Delivery

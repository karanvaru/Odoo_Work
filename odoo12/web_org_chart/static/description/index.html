Organization charts are effective, transparent and they communicate a lot about an organization in just a single glance. This module is compatible with Odoo 13 Community and Enterprise Editions. an org chart makes it easier for people to sunderstand a large amount of information as a visual picture rather than as a table of names and numbers.

 
 

An organization chart is a graphical description of the roles and structure within an organization. The charts also diagram the relationship between those roles and they are a simple way to visualize how workflows within a business.

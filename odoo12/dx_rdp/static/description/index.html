Create Employee And Users

There are four roles involved Employee, Line manager, Hr manager, And CEO.

Create user and employee for the employee, line manager, hr manager, and CEO.


  • Here relate the user-created before in the user section for employees after that you have to do the same thing for a line manager, hr manager, and CEO.


    • Create Employee Disciplinary

    Disciplinary can be created by only line manager.



    The line manager will create a disciplinary and submit it on submission, it will send a notification to the hr manager, who will review it and either approve or disapprove it.


    If hr manager approves this disciplinary notice then the line manager and employee will be notified after that line manager will create a performance improvement plan (PIP).



    The employee will either approve and accept the pip. if he accepts it then notification will be sent line manager and hr manager and disciplinary is closed. If not accepted by the employee then the employee has the right to appeal. employee raises an appeal and a notification will be sent to the hr manager.




    When the hr manager gets this appeal then he can accept or refuse if the hr manager accepts it then he will submit this appeal to the CEO. If hr manager refuses this appeal then the employee has to accept PIP.


    When the CEO gets this application then he can also upheld or refuse the appeal. if the CEO uphelds this appeal then hr manager and employee are notified. then a mutual agreement or discussion takes place and the disciplinary is sorted out. If CEO refuses this application then the employee and hr manager will be notified and the employee has to accept PIP.

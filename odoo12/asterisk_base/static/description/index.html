Asterisk PBX Manager

Ready to run Asterisk PBX managed from Odoo!

Features

  • Fully functional Asterisk management:
    • Advanced WEB based asterisk .conf editor with asterisk syntax highlight.
    • Extensions management.
    • Custom dialplan management.
    • Security - blocking password attempts.
  • Batteries included: Basic configuration with SIP users, company menu, sales & support queues, and more...
  • Full Asterisk WEB console (CLI).
  • Multi server support - manager all your Asterisk servers from one Odoo instance!
  • Loosely coupled architecture - an Asterisk server (bare metal or virtual) is provisioned from Odoo and does not depend on Odoo afterwards.
  • Get a brand new and ready to use PBX with no hustle or connect your existing Asterisk installation and start migration without any interruption.
  • More features to come: conference manager, menu builder, and more and more and more.... ;-)

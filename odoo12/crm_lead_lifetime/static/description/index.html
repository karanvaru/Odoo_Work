If you are using CRM then you know how it’s important to manage the pipelines. CRM can have any number of custom and dynamic pipelines. In these pipelines, we take care of leads.

"This plugin is one of the best components that give us the ability to find the lead’s lifetime in each stage. No matter if the stages are dynamic or custom."


The plugin easily access data and get integrated in few steps. For every stage you can find out the number of days the lead resides into each stage. For example, if the lead is just created and it’s in a new stage. After some days you switch next to the new stage then, the leads lifetime in the number of days at the new stage will be displayed. Not only displays the age of lead at each stage but also you can get a report. Users can apply the filters and get the desired output.

No additional configuration is required.

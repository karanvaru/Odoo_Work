Sale Priority

This plugin will help to add priority in Quotation / Sale Order.



Features

Easy to give priority to sale quotation or sale order.

Filter sale quotation or sale order by priority.

Group by sale quotation or sale order by priority.

How Application Work



Priority in Sale Quotation Form.


Priority in Sale Tree View.



Priority in Sale kanban view.


Group By Priority..


Filter Sale by priority Low, Medium, High, Very High

This Module Add Feature To Pass Field Value One Object To Another!

Key Features
  • Add Custom Field Value From Sale Order To Invoice, Delivery, Journal Entry, Journal Item
  • Add Custom Field Value From Purchase Order To Invoice, Delivery, Journal Entry, Journal Item
  • If Change Value From One Form All Forms Field Value Will Change
  • Hide Create,Edit And Delete Button On Product And Product Variant Page Base On Security Group
1. Open Menu Sales > Orders > Orders
2. Create New Order, Add Customer, Order Lines, Record Type And Record Category Value
3. Confirm Order And Open Delivery Order
4. Record Type And Record Category Field Value Will Automatically Set, Validate Order
5. Open Menu Inventory > Reporting > Stock Moves
6. Open Delivery Order's Move, Value Will Set
7. Create Invoice Form Sale Order
8. Open invoice
9. Record Type And Record Category Field Value Will Set From Sale Order, Validate Invoice
10. Open Menu Invoicing > Accounting > Journal Entries
11. Open New Created Journal Entries, Field Value Set From Invoice
12. Open Sale Order And Change Value For Record Type Or Record Category
13. Also Change From Delivery Order
14. Also Change From Stock Move
15. Also Change From Invoice
16. Also Change From Journal Entries
17. Open Menu Purchase > Purchase > Purchase Order
18. Create New Order, Add Customer, Product, Record Type And Record Category
19. Confirm Order And Open Receipt
20. Record Type And Record Category Fields Value Set Form Purchase Order.
21. Open Menu Inventory > Reporting > Stock Moves,
Open New Generated Moves, Set Value From Receipt
22. Create Vendor Bill From Purchase Order And Open Bill
23. Value Set Form Purchase Order, Validate Bill
24. Open Menu Invoicing > Accounting > Journal Entries
Open New Created Entry, Value Will Set From Vendor Bill And Also Set In Journal Items
When Change It In Any One Form It Will Change In All Form, Similar To sale
25. Open Menu User & Companies > Users
26. Give Administration > Settings Rights For User
27. Login With That User
Open Menu Inventory > Product Variants, Create And Delete Button Will Visible
28. In Form View Buttons Are Visible
29. Open Product Page In There Also Buttons Visible
30. In Kanban View Create Button Is Visible
31. in Form View Create,Edit And Delete Button Are Show
32. Remove Settings Rights From User
33. And Then After Check, Buttons Will Hide
From Product Tree View
34. From Product Form View
35. From Product Variant Tree View
36. From Product Variant Kanban View
37. From Product Variant From View

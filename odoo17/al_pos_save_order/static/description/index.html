POS Save Order

Key Features
  • Save Order In Pending Sales List .
1. Open Menu Point of Sale > Dashboard And Click On Continue Selling.
2. Add Product And Save Order And Open Pending Sales List.
3. Generated Order Will Save In List.
4. Add Other Product For New Order.
5. New Order Will Added In Pending Sales List.
6. Save Order Without Add Product.
7. Warning Will Show.

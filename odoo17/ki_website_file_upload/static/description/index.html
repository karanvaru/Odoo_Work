Upload Attachment On Website Sales

This Module Allow To Attach Documents To Sale While Placing The Order!

Key Features
  • Attach Multiple Documents While Placing Order
  • Uploaded Documents From Ecommerce Order Will Be Stored On Sale Order
1. Open Menu Website > Shop And Select Product And Check Out It
2. Click On Add Attachment.
3. Select Document To Be Uploaded
4. Stored Attachments Will Be Displayed, Click On Hide/Show Attachment If You Want To Show Or Hide Attachment
5. Uploded Document From E-commerce Is Stored On Sale Order
6. Uploaded Documents Are Stored As Attachments

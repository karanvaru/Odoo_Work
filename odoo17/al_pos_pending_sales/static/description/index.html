POS Pending Sales

Key Features
  • Add Pending Sales Button.
1. Open Menu Point of Sale > Dashboard And Click On Continue Selling.
2. Add Product And Open Pending Sales View.
3. Added Product Will Show Here Generate New Order.
4. Add Other Product For Other Order And Again Open Pending Sales View.
5. New Generated Order And Pending Order Will Show, Select Order And Load Order.
6. Selected Order Will Continue, Generate Payment.
7. Select Payment Method And Validate Order.
8. Payment Successfully, New Order And Resume Order Option Will Show.
9. Open Resume Order It Will Redirect Pending Order List.
10. If Select New Order Then Add New Product And Check Pending Sales List.
11. Pending Order Will Save Here And New Order Also Set.

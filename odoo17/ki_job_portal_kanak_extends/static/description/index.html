Usage Of Module
1. Open Menu Recruitment > Configuration > Functional Area And Create Record
2. Open Menu Recruitment > Configuration > Department And Create Department
3. Open Menu Recruitment > Configuration > Application Document Type
4. Create New Document
Is Default :- Check If You Want To Set Document In Website By Default
Is Required :- Check If You Want To Set Document Is Required
5. Open Menu Website > Post Job
6. Create New Job
7. Open Menu Website > Job
Open New Created Job
8. Apply For Job
9. Fill Applicant Information And Click On Next
10. Fill Applicant Address And Click On Next
11. Fill Academic Details And Click On Next
12. Fill Certificate details And Click On Next
13. Fill Professional details And Click On Next
14. Fill Application Summary And Click On Next
15. Fill Trades Assessment Questionnaire And Click On Next
16. Upload Your Document And Click On Next
17. Open Menu Recruitment > By Job Position And Open New Created Position
18. Newly Created Record Will Display Open Detail View
19. Open Detail View
20. Data Set In All Tab
21. Open Menu Recruitment > configuration > Recruitment Agent
22. Agent Will Show In Website
23. Selected Agent Set In Application

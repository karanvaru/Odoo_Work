Reddot Contract Extension

Key Features
  • Generate Contract Report Based On Dynamic Template.
  • Additional Features Include:
    • Accept Or Reject Contracts.
    • Raise Exceptions For Contracts.
    • Renew Or Cancel Contract Renewals.
    • Send Email Notifications Related To Contracts.
1. Open Menu Settings > Technical > Scheduled Action
2. Open 60 Days Reminder Cron And Run It
3. Open Menu Settings > Technical > Emails
4. Email Will Send Employees Manager Which Employee Contract Will Terminate In 60 Days
5. And Employees Contract Status Will Be In To Renew
6. Renew Contract
7. It Will Forward In To Approve Stage
8. It Will Send Email To Employees Department Manager When Contract In To Approve
9. Approve Contract It Will Open PopUp Add Details And Submit It
10. Renew Contract With New Date And Pass Comment
11. Old Contract Will Approved
12. If Do Not Renew Contract Then
13. Add Remark And Submit Popup
14. Contract In Do Not Renew State And Remark Will Added
15. Send Contract Mail And Set State To Sent
16. Send It
17. Mail Send With Body
18. View Attachment
19. In Case Of Accept Contract
20. Add Signature And Submit
21. Signature Will Add In New Tab
22. In Case Of Exception
23. Add Remark And Submit It
24. It Will Store In Exception Remark
25. Open Menu Employee > Configuration > Contract Template
26. Add Template Name Company And Other Value Inside {} Which Is Field Name Which Want To Print In Pdf
27. Add Report Template Value Inside Contract And Generate Report Data
28. Data Will Generate Based On Employee Contract

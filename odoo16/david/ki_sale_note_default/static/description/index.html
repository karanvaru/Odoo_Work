Sales Default Note Template

Key Features
  • Define multiple Templates of Sale Default Note.
  • We can define template based on customers also.
  • Select template on sale order according to customer and it will automatically default set Note on sale order Line.
  • If customer is defined on template then that template only available for specific customer only.

2. Click on New.
3. Sale Default Note Template Details.
4. Sale Default Note Template on Sale Order.

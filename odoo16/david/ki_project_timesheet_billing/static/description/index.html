Project Timesheet Billing

Key Features
  • Create Invoice Based On Employee Timesheet And Hour/Rent
Usage Of Module
1. Open Menu Settings > Project
And Add Product In TimeSheet Billing Product
2. Open Menu Job Card > Job Card And Create New Card
3. Add Project
4. Open Project
Add Billing Employee Rate In Tab
5. Add Invoice Line In Job Invoice Line
6. Add Timesheet In Timesheet Tab
7. Add Details And Create Invoice
8. If Employee Of Timesheet Not In Project Billing Employee Rate Then Give Warning
9. Add That Employee In Project Billing Employee Rate ,Then Create Invoice Again
10. Invoice Will Create, Open Invoice
11. Invoice Line And Timesheet Data Will Set In Invoice Lines
12. Add Timesheet Again
13. Add Other Invoice Line And Create Invoice Again
14. Second Invoice Will Create For Card
15. Open Invoice
Invoice Will Create For New Invoice Line And Timesheet

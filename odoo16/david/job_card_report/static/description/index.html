Job Card Analysis Report

Job Card Reports Odoo App

Job Card App Reporting in Odoo for Workshops / Factory / Any Technical / Repair Shops / Industry

This app allows all your project managers to view job card analysis reports as shown the below screenshots.


Main Features
  • Report 1: Job Card Instructions Report with Pivot View, Graph View and List View.
  • Report 2: Job Card Costsheets Report with Pivot View, Graph View and List View.
  • Report 3: Job Card Requisitions Lines Report with Pivot View, Graph View and List View.
  • Report 4: Job Card Timesheets Report with Pivot View and List View.
  • For more details please see the video and check below screenshots.
  • You can check dependent job_card app by clicking below link:https://apps.odoo.com/apps/modules/16.0/job_card/

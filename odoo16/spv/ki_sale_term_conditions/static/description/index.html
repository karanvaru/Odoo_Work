Sales Term and Condition Templates

This module allow define Multiple Templates for Term and Conditions and use it on sale order!

Key Features
  • Define multiple Templates of Term and Conditions
  • We can define template based on customers also
  • Select template on sale order according to customer and it will automatically set Term and Condition on sale order
  • If customer is defined on template then that template only available for specific customer only

CRM Extension

Key Features
  • Opportunity Wise Sale Order Confirm Or Cancelled
  • Based On Sale Order Confirm Opportunity Will Won
1. Open Menu Sales > Order And Create Order And Set Value For Opportunity
2. Open Menu CRM > Pipeline, Open Opportunity Which Opportunity Set In sale Order Click On Mark Won Button
3. Pop Up Will Open Select Sale Order And Submit It
4. Open Sale Order It Will Confirm
5. Opportunity Stage Will Won And Sale Order Amount Display
6. Create Sale Order
7. Open Menu CRM > Pipeline, Open Opportunity Which Opportunity Set In Sale Order Click On Mark Lost Button
8. Pop Up Will Open Select Sale Order And Reason For Lost And Submit It
9. Opportunity Will Lost
10.Open Sale Sale Order It Will Be Cancelled
11. Create New Sale Order Set Opportunity And Confirm Order
12. Opportunity Will Set As Won And Sale Order Amount Will Display

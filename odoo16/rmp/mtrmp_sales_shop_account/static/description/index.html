This Module Add Feature Related To Accounting !

Key Features
  • Create Payment From Payment Settlement
  • Create Invoice When All Picking Are Done
  • Create Credit Note When Return Picking Are Done
1. Open Menu Sale > Order > Quotation And Create New Quotation
2. Fill Details And Confirm Quotation
3. Delivery Order Will Create Open It From Smart Button
4. Open Picking Record Add Quantity And Validate It
5. Open Second Record And Validate It
6. Open Last Record And Validate It
7. Apply Quantities For Order
When Last Picking Will Done Then Invoice Will Create
8. Open Menu Accounting > Customer > Invoices And Open New Created Invoice
9. Open Last Done Picking And Return It
10. Add Quantities And Return It
11. New Return Picking Order Will Create
12. Open Returned Picking And Apply Quantities And Validate It
When Last Return Order Will Done ,Credit Note Will Create And invoice Order Will Confirm
13. Customer Invoice
14. Open Menu Accounting > Customer > Credit Notes And View New Created Note
15. Open Menu Sales > Shop/Sales Channel > Shop
16. Open Payment Channel
17. Accounting Configuration,
Set Value For Journal And Payment Method, Add Charges Type Which From Payment Settlement And Add Account In Which Account Payment Will Create
18. Open Payment Settlement
19. Create Invoice If Not Any In Invoice In This Order
20. Create Invoice And Confirm It.
21. Set Value For Payment
22. Run Server Action, New Payment Will Create
23. Open Menu Accounting > Customer > Payment,Open NEw Created Payment
24. Open Journal Entry Button
26. Account Wise Field Value Credit/Debited
27. You Can Also Generate It From Tree View

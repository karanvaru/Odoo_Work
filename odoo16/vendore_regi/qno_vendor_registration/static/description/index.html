Vendor Registration

Vendor Registration From Website!

Key Features
  • Send Mail To User
  • User Add His Details Like
    • Primary Details
    • Company Pan Details
    • Company GST Details
    • Company Address
    • Bank Details
  • Approve Or Reject User
1. Open Menu Vendor Registration > Configuration > Organization Type
 Create New Organization Type If Required
2. Open Menu Vendor Registration > Configuration > Business Category
 Create New Business Category
3. Open Menu Vendor Registration > Configuration > Settings
4. Here Add Value For URL, Api Url, Database Name And Active Or Not
5. Open Menu Waiting For Approval
6. Generate New Record And Send Mail
7. Open Menu User & Companies > Companies
8. Add Company Logo And Email Background Image
9. Open Menu Technical > Emails
10. Email Will Sent
11. Click On Accept Invite
12. Web Page Will Open

Validation For Mobile Number
Add Child User For Current User
Confirm & Process
12. Company Pan Details
Add Related Details
Validation For PAN Number Format
Confirm & Process

Add Related Details
Validation For GST Number

Confirm & Process

Confirm & Process

Add Bank Details
Raise Warning If Account Number And Re-entered Account Number Will Not Same
Check Box To Add Other Bank Account
17. Add Other Bank Account Details
Confirm & Process

19. Open That User, Details Fill From Website
20. Bank Details Will Set

22. Company Details Will Set
